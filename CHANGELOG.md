# Changelog

## v1.0.0 - 2026-06-20

### Korean

- Windows 포터블 앱 첫 배포
- Tesseract 및 한국어, 영어, 한자 OCR 데이터 포함
- 원본 이미지 보존형 검색 가능 PDF 생성
- 그림 영역 보수적 제외와 본문 재확인 기능
- 진행률, 경과 시간, 예상 남은 시간 표시

### English

- First portable Windows release
- Bundled Tesseract with Korean, English, and Hanja OCR data
- Searchable PDF generation that preserves original images
- Conservative graphic exclusion with body-text recovery checks
- Progress, elapsed-time, and ETA display
