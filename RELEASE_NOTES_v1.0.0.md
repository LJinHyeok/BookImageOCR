## 한국어

PDF에서 선택이 안 되는 글자들을 선택할 수 있도록 만들어주는 첫 Windows 포터블 배포입니다. 스캔 PDF의 본문 글자와 사진·도표·삽화 영역을 구분해 본문에만 검색 가능한 OCR 텍스트 레이어를 추가하고, 원래 페이지 이미지는 그대로 유지합니다.

- 한국어, 영어, 한자 OCR 포함
- Python 및 Tesseract 별도 설치 불필요
- 차트와 삽화는 보수적으로 제외하고 본문 누락은 재확인
- 진행률, 경과 시간, 예상 남은 시간 표시

## English

The first portable Windows release makes previously unselectable PDF text selectable. It separates body text from photos, charts, and illustrations in scanned PDFs, adds a searchable OCR text layer to the body text, and preserves the original page image.

- Korean, English, and Hanja OCR included
- No separate Python or Tesseract installation required
- Conservative graphic exclusion with body-text recovery checks
- Progress, elapsed-time, and ETA display

Download the Windows ZIP, extract it, and run `BookImageOCR.exe`.
